# Estatística Aplicada com Python

## 📌 Sobre o repositório
Este repositório reúne atividades desenvolvidas na disciplina de Estatística Aplicada, utilizando Python e Jupyter Notebook para análise de dados e visualização de resultados.

Cada atividade é organizada por aula, contendo o notebook com a análise e os dados utilizados.

---

## 🎯 Objetivo
Aplicar conceitos estatísticos na prática por meio da análise de dados, explorando técnicas de interpretação e visualização.

---

## 📊 Conteúdos abordados
- Estatística descritiva
- Medidas de tendência central e dispersão
- Análise exploratória de dados (EDA)
- Visualização de dados

---

## 🛠️ Tecnologias e bibliotecas
- Python
- Jupyter Notebook
- Pandas
- NumPy

---

## 📁 Estrutura do repositório

## ▶️ Como executar

1. Instale as dependências:
`pip install pandas numpy`

2. Abra os notebooks:
`jupyter notebook`

3. Execute as células para visualizar as análises
